# JSON Parsing Assignment

## Overview

This project demonstrates JSON parsing and manipulation using Python.

The assignment involved reading data from a JSON file, extracting information, answering a set of 20 questions, and implementing different JSON handling methods provided by Python's `json` module.

## Concepts Covered

* JSON Parsing
* Python Dictionaries and Lists
* File Handling
* Loops and Conditional Statements
* Exception Handling
* Data Aggregation and Transformation

## JSON Methods Implemented

### json.loads()

Used to convert JSON data stored as a string into a Python dictionary.

### json.dump()

Used to write Python data into a JSON file (`output_data.json`).

### json.dumps()

Used to convert Python data into a JSON-formatted string and store it in a variable or text file (`answers_string.txt`).

## Project Files

* `data_dump.json` - Input JSON data
* `main.py` - Main program containing solutions to all 20 questions
* `output_data.json` - Generated JSON file containing question-answer pairs
* `answers_string.txt` - JSON data converted into string format using `json.dumps()`
* `json_methods_demo.py` - Demonstration of JSON methods

## Output

The program:

1. Reads and parses the input JSON file.
2. Answers all 20 questions.
3. Stores the answers in a dictionary.
4. Generates `output_data.json` using `json.dump()`.
5. Generates a JSON string using `json.dumps()`.
